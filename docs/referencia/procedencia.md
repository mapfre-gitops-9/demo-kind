# Procedencia y revisión del material

Este repositorio procede de sesiones internas de divulgación sobre
contenedores, redes, Docker, Kubernetes y formas de trabajo colaborativas.

## Fuentes y herramientas

- Los recaps iniciales de las sesiones se generaron con Microsoft 365 Copilot.
- Parte del código y del material elaborado posteriormente se trabajó con
  Antigravity.
- La revisión integral del material producido previamente con Antigravity se
  realizó con Codex, usando GPT-5.5 y razonamiento Extra High (`xhigh`).

## Criterio editorial

Los recaps automáticos se consideran fuentes auxiliares, no actas verificadas.
No se publican directamente porque pueden contener errores, atribuciones
personales o contexto que no pertenece al objetivo formativo.

Para elaborar el contenido vigente se aplican estos controles:

1. Contrastar conceptos con documentación primaria.
2. Verificar comandos, builds, pruebas y manifiestos.
3. Eliminar nombres, datos personales y detalles internos innecesarios.
4. Separar las simplificaciones de la maqueta de las exigencias de producción.
5. Someter los cambios a revisión humana y controles automáticos.

El uso de IA no convierte el contenido en correcto ni sustituye la
responsabilidad de quienes lo mantienen.

La configuración `xhigh` corresponde al nivel Extra High de esfuerzo de
razonamiento documentado para Codex.
